# Nuzzle_Frontend

**UMC 너즐 프로젝트 프론트엔드 레포지토리**

## <b> 0. Installation </b>

- `Clone` repository

```shell
$ git clone <repo-url>
```

- `Change directory` to cloned repository

```shell
$ cd <repo>
```

- Install `package`

```shell
$ npm install
```

- Start project

```shell
$ npm run dev
```

## <b> 1. Our Branch Strategy </b>

1. `main` : CI/CD
2. `dev`: dev branch
3. `merge-branch` : merge branch
4. `feature/fill simple description`: development new feature
